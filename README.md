## dependencies
- python3.10

## Commands
### show
#### flags
- --list/-l <list name> (required)
- --state/-s <0/1> (optional)

The command shows the selected List.

### add
#### flags
- --list/-l <list name> (required)

If the selected list exists it will add the task, else it will create the list and add the task to this list.
